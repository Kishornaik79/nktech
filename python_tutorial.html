Syntax & Print

Python's syntax is clean and minimal. No curly braces, no semicolons ‚Äî just clean, readable code. Indentation (spaces) is how Python groups code blocks.

hello.py
# This is a comment ‚Äî Python ignores it

# Print output to the screen
print("Hello, World!")
print("Welcome to NkTechVision!")

# Print multiple values
print("Name:", "Kishor", "Age:", 25)

# f-strings ‚Äî modern, clean way to format strings
name = "Kishor"
print(f"Hello, {name}! Welcome to Python.")

# Print with sep and end
print("Python", "is", "awesome", sep="-")  # Python-is-awesome
print("No newline", end=" ")
print("same line")
üí° Indentation is Everything in Python

Python uses 4 spaces (or 1 tab) of indentation to define code blocks ‚Äî inside if statements, loops, functions, and classes. Unlike JavaScript, forgetting indentation causes an error!

Variables & Data Types

Python is dynamically typed ‚Äî you don't need to declare the type of a variable. Python figures it out automatically.

types.py
# String ‚Äî text
name = "NkTechVision"
greeting = 'Hello!'         # single or double quotes
multi = """This spans
multiple lines."""

# Integer ‚Äî whole number
age = 25
year = 2025

# Float ‚Äî decimal number
price = 9.99
pi = 3.14159

# Boolean ‚Äî True or False (capital letters!)
is_free = True
has_ads = False

# None ‚Äî like null in other languages
result = None

# Check the type
print(type(name))   # <class 'str'>
print(type(age))    # <class 'int'>

# Type conversion
num_str = "42"
num_int = int(num_str)  # "42" ‚Üí 42
num_flt = float(age)   # 25 ‚Üí 25.0

If / Elif / Else

Python uses if, elif (else if), and else to make decisions. Notice: no parentheses needed around the condition, and end with a colon :

conditions.py
score = 78

if score >= 90:
    print("Grade: A ‚Äî Excellent!")
elif score >= 70:
    print("Grade: B ‚Äî Good job!")   # This runs
elif score >= 50:
    print("Grade: C ‚Äî Pass")
else:
    print("Grade: F ‚Äî Try again")

# Multiple conditions with and / or
age = 20
has_id = True
if age >= 18 and has_id:
    print("Access granted")

# One-line ternary (conditional expression)
status = "Adult" if age >= 18 else "Minor"
print(status)  # Adult

Loops

Python has two main loop types: for loops (iterate over a sequence) and while loops (repeat while a condition is true).

loops.py
# FOR loop with range()
for i in range(1, 6):     # 1 to 5
    print(f"Count: {i}")

# FOR loop over a list
languages = ["Python", "JavaScript", "Java"]
for lang in languages:
    print(f"Language: {lang}")

# enumerate() ‚Äî get index AND value
for i, lang in enumerate(languages, 1):
    print(f"{i}. {lang}")

# WHILE loop
count = 0
while count < 3:
    print(f"count = {count}")
    count += 1

# break and continue
for num in range(10):
    if num == 3: continue  # skip 3
    if num == 7: break    # stop at 7
    print(num)

# List comprehension ‚Äî powerful one-liner loop
squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]

Lists

A Python list is an ordered, changeable collection of items. It can hold any mix of data types.

lists.py
courses = ["HTML", "CSS", "JavaScript", "Python"]

# Access by index (starts at 0, -1 = last)
print(courses[0])   # "HTML"
print(courses[-1])  # "Python"

# Slicing ‚Äî get a portion
print(courses[1:3])  # ["CSS", "JavaScript"]

# Common methods
courses.append("Java")      # add to end
courses.insert(0, "English") # insert at index
courses.remove("CSS")       # remove by value
courses.sort()               # sort alphabetically
courses.reverse()           # reverse order
print(len(courses))          # number of items

# Check if item exists
if "Python" in courses:
    print("Python is in the list!")

Dictionaries

A dictionary stores data as key: value pairs ‚Äî like a real dictionary where you look up a word (key) to get its meaning (value).

dicts.py
student = {
    "name": "Kishor",
    "age": 25,
    "courses": ["Python", "JavaScript"],
    "enrolled": True
}

# Access values
print(student["name"])         # Kishor
print(student.get("age"))      # 25 (safe access)
print(student.get("grade", "N/A"))  # N/A (default)

# Add / update
student["city"] = "Mumbai"
student["age"] = 26

# Loop through dictionary
for key, value in student.items():
    print(f"{key}: {value}")

# Check if key exists
if "name" in student:
    print("Name found!")

# Delete a key
del student["enrolled"]

Functions

Functions let you write code once and reuse it anywhere. In Python, define a function with def.

functions.py
# Basic function
def greet(name):
    print(f"Hello, {name}! Welcome to NkTechVision.")

greet("Kishor")

# Function with return value
def add(a, b):
    return a + b

result = add(10, 25)
print(result)  # 35

# Default parameters
def power(base, exponent=2):
    return base ** exponent

print(power(5))      # 25  (5¬≤)
print(power(2, 10)) # 1024 (2¬π‚Å∞)

# *args ‚Äî accept any number of arguments
def total(*nums):
    return sum(nums)

print(total(1, 2, 3, 4, 5))  # 15

# Lambda ‚Äî one-line anonymous function
square = lambda x: x ** 2
print(square(7))  # 49

Classes & OOP

Object-Oriented Programming (OOP) organises code around objects ‚Äî bundles of data and behaviour. A class is a blueprint; an object is an instance of that blueprint.

oop.py
class Student:
    """A class representing a student"""

    def __init__(self, name, age):
        # Instance variables
        self.name = name
        self.age = age
        self.courses = []

    def enrol(self, course):
        self.courses.append(course)
        print(f"{self.name} enrolled in {course}!")

    def info(self):
        print(f"Name: {self.name}, Age: {self.age}")
        print(f"Courses: {', '.join(self.courses)}")

# Create objects (instances) from the class
s1 = Student("Kishor", 25)
s2 = Student("Priya", 22)

s1.enrol("Python")
s1.enrol("JavaScript")
s1.info()

# Inheritance ‚Äî one class extending another
class GraduateStudent(Student):
    def __init__(self, name, age, thesis):
        super().__init__(name, age)  # call parent
        self.thesis = thesis

    def info(self):
        super().info()
        print(f"Thesis: {self.thesis}")
üêç Python OOP Key Concepts

self refers to the current instance. __init__ is the constructor ‚Äî runs when you create an object. super() calls the parent class. Inheritance lets child classes reuse parent code.
