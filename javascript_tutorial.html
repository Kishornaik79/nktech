What is JavaScript?

JavaScript (JS) is the programming language of the web. While HTML creates structure and CSS adds style, JavaScript makes pages interactive and dynamic.

Every time you click a button that opens a menu, submit a form without reloading the page, or see a live clock ‚Äî that's JavaScript at work. It runs directly in the browser, no installation needed.

‚ö° Why JavaScript?

JavaScript is the only language that runs natively in every web browser. It's also used on servers (Node.js), in mobile apps, and even AI tools. Learning JS unlocks the entire web.

Variables

A variable is a named container that stores a value. Think of it like a labelled box ‚Äî you put something in it and refer to it by name later.

script.js
// let ‚Äî can be changed later (use this most often)
let name = "Kishor";
let age = 25;

// const ‚Äî cannot be changed (use for fixed values)
const siteName = "NkTechVision";
const pi = 3.14159;

// var ‚Äî old style, avoid in modern JS
var oldWay = "Don't use this";

// Using variables
console.log(name);        // Kishor
console.log(age + 1);     // 26
console.log("Hello, " + name); // Hello, Kishor
üí° let vs const ‚Äî Quick Rule

Default to const. If you need to reassign the value later (like a counter or user input), use let. Never use var in modern JavaScript.

If / Else Statements

Conditionals let your code make decisions ‚Äî run one block if something is true, and a different block if it's false.

script.js
let score = 75;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 70) {
  console.log("Grade: B");   // This runs
} else if (score >= 50) {
  console.log("Grade: C");
} else {
  console.log("Failed");
}

// Ternary operator ‚Äî short if/else in one line
let result = score >= 50 ? "Pass" : "Fail";
console.log(result);  // Pass

Loops

Loops repeat a block of code multiple times ‚Äî great for processing lists, counting, or doing repetitive tasks.

script.js
// FOR loop ‚Äî when you know how many times to repeat
for (let i = 1; i <= 5; i++) {
  console.log(`Count: ${i}`);
}
// Count: 1, Count: 2, ... Count: 5

// WHILE loop ‚Äî repeat while condition is true
let n = 0;
while (n < 3) {
  console.log("n = " + n);
  n++;
}

// FOR...OF ‚Äî loop through an array easily
const fruits = ["apple", "banana", "mango"];
for (const fruit of fruits) {
  console.log(fruit);
}

Functions

A function is a reusable block of code. Write it once, run it anywhere. Functions are the building blocks of every JavaScript program.

script.js
// Function declaration
function greet(name) {
  return `Hello, ${name}! Welcome to NkTechVision.`;
}

console.log(greet("Kishor"));  // Hello, Kishor! Welcome...
console.log(greet("Student")); // Hello, Student! Welcome...

// Arrow function ‚Äî modern, shorter syntax
const add = (a, b) => a + b;
console.log(add(5, 3));  // 8

// Function with default parameter
const power = (base, exp = 2) => base ** exp;
console.log(power(4));     // 16  (4¬≤)
console.log(power(2, 10)); // 1024 (2¬π‚Å∞)

Arrays

An array is an ordered list of values. Arrays are one of the most useful structures in JavaScript.

script.js
const courses = ["HTML", "CSS", "JavaScript", "Python"];

// Access by index (starts at 0)
console.log(courses[0]);  // "HTML"
console.log(courses[2]);  // "JavaScript"

// Common array methods
courses.push("Java");       // Add to end
courses.pop();              // Remove from end
courses.unshift("English"); // Add to start

console.log(courses.length);  // Number of items

// Powerful array methods
const nums = [1, 2, 3, 4, 5];

// map ‚Äî transform each item
const doubled = nums.map(n => n * 2);  // [2,4,6,8,10]

// filter ‚Äî keep only matching items
const evens = nums.filter(n => n % 2 === 0); // [2,4]

// find ‚Äî first item that matches
const big = nums.find(n => n > 3);  // 4

Objects

An object groups related data together using key: value pairs. Perfect for representing real-world things like a user, a product, or a course.

script.js
const student = {
  name: "Kishor",
  age: 25,
  courses: ["HTML", "CSS", "JavaScript"],
  isEnrolled: true,
  // Functions inside objects are called methods
  greet: function() {
    return `Hi, I'm ${this.name}!`;
  }
};

// Access properties
console.log(student.name);       // "Kishor"
console.log(student["age"]);    // 25
console.log(student.greet());   // "Hi, I'm Kishor!"

// Add or update a property
student.city = "Mumbai";

// Destructuring ‚Äî extract into variables
const { name, age } = student;
console.log(name, age); // Kishor 25

The DOM

The DOM (Document Object Model) is how JavaScript sees and interacts with your HTML page. Using the DOM you can read, change, add, and remove HTML elements with JavaScript.

script.js
// SELECT elements
const title = document.getElementById("title");
const btn = document.querySelector(".btn");      // first .btn
const all = document.querySelectorAll("p");       // all <p> tags

// CHANGE content
title.textContent = "New Title!";
title.innerHTML = "<strong>Bold Title</strong>";

// CHANGE styles
title.style.color = "gold";
title.style.fontSize = "2rem";

// ADD/REMOVE CSS classes
btn.classList.add("active");
btn.classList.remove("hidden");
btn.classList.toggle("dark");  // adds if missing, removes if present

// CREATE new elements
const newPara = document.createElement("p");
newPara.textContent = "I was added by JavaScript!";
document.body.appendChild(newPara);

Events

Events are things that happen in the browser ‚Äî a user clicks, types, scrolls, or moves their mouse. JavaScript can listen for these events and react to them.

script.js
const btn = document.querySelector("#myButton");

// addEventListener(event, function)
btn.addEventListener("click", function() {
  alert("Button clicked!");
});

// Common events
btn.addEventListener("mouseover", () => { /* hover */ });
btn.addEventListener("mouseout",  () => { /* leave */ });

const input = document.querySelector("input");
input.addEventListener("input", (e) => {
  console.log("You typed: " + e.target.value);
});

document.addEventListener("keydown", (e) => {
  console.log("Key pressed: " + e.key);
});

function updateCounter(change, reset = false) {
  if (reset) { count = 0; }

  display.textContent = count;
  display.style.color = count > 0 ? "#7fffb2"

Random Color Generator

Click the button to generate a random hex color ‚Äî uses Math.random() and DOM manipulation.

#333333
color.js
function randomColor() {
  // Generate random hex like #a3f2b1
  const hex = "#" + Math.floor(Math.random() * 0xFFFFFF)
    .toString(16)
    .padStart(6, "0");

  const box = document.getElementById("colorBox");
  box.style.background = hex;
  box.textContent = hex;

  // Auto-pick text color for readability
  const r = parseInt(hex.slice(1,3), 16);
  const g = parseInt(hex.slice(3,5), 16);
  const b = parseInt(hex.slice(5,7), 16);
  const brightness = (r * 299 + g * 587 + b * 114) / 1000;
  box.style.color = brightness > 128 ? "#000" : "#fff";
}
