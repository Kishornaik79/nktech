What is CSS?

CSS stands for Cascading Style Sheets. While HTML creates the structure of a page (the skeleton), CSS controls how it looks â€” colours, fonts, spacing, layout, and animations.

Without CSS, every webpage would just be plain black text on a white background. CSS is what makes the web beautiful.

ðŸ’¡ The "Cascade" in CSS

CSS rules "cascade" â€” meaning when multiple rules apply to the same element, the browser follows specific priority rules to decide which one wins. More specific rules beat less specific ones.

Three Ways to Add CSS

index.html 
/* 1. External CSS (BEST â€” put in a separate .css file) */

/* 2. Internal CSS (inside <style> tag in <head>) */

  h1 { color: blue; }
</style>

/* 3. Inline CSS (directly on the element â€” avoid if possible) */

Syntax & Selectors

A CSS rule has two parts: a selector (what to style) and a declaration block (how to style it).

style.css
/* Selector { property: value; } */

h1 {
  color: purple;
  font-size: 32px;
}

/* Class selector (.) â€” reusable on many elements */
.highlight {
  background-color: yellow;
  padding: 4px 8px;
}

/* ID selector (#) â€” unique, use only once per page */
#hero {
  background: navy;
  color: white;
}

/* Multiple selectors at once */
h1, h2, h3 {
  font-family: Georgia, serif;
}

/* Hover state â€” triggers when mouse is over element */
button:hover {
  background-color: darkblue;
}

Text & Fonts

Typography is one of the most powerful tools in design. CSS gives you full control over fonts, sizes, spacing, and alignment.

style.css
body {
  font-family: 'Georgia', serif;    /* Font choice */
  font-size: 16px;                /* Base size */
  font-weight: 400;               /* Normal weight */
  line-height: 1.6;              /* Space between lines */
  color: #1a1410;
}

h1 {
  font-size: 2.5rem;             /* rem = relative to root */
  font-weight: 800;               /* Bold */
  letter-spacing: -1px;         /* Tighten letters */
  text-align: center;
}

p {
  text-transform: uppercase;    /* ALL CAPS */
  text-decoration: underline;  /* Underline */
  text-indent: 2rem;            /* First line indent */
}

The Box Model

Every HTML element is a box. Understanding the box model is the single most important concept in CSS layout. Each box has 4 layers:

Margin â€” space outside
Border â€” the edge
Padding â€” space inside
Content (text, imageâ€¦)
style.css
.card {
  /* Content size */
  width: 300px;
  height: 200px;

  /* Padding: space INSIDE the box */
  padding: 20px;             /* all sides */
  padding: 10px 20px;       /* top/bottom  left/right */

  /* Border */
  border: 2px solid #a855f7;
  border-radius: 12px;       /* Rounded corners */

  /* Margin: space OUTSIDE the box */
  margin: 24px auto;        /* auto = center horizontally */

  /* box-sizing: border-box is a game changer */
  box-sizing: border-box;   /* padding included in width */
}
ðŸ”‘ Always Use box-sizing: border-box

Add * { box-sizing: border-box; } at the top of every CSS file. This makes widths behave predictably â€” padding won't add unexpected extra size to your elements.

Flexbox

Flexbox is the most useful layout tool in modern CSS. It makes it easy to align, space, and reorder items in a row or column â€” with just a few lines of code.

style.css
.container {
  display: flex;

  /* Direction: row (default) or column */
  flex-direction: row;

  /* Horizontal alignment */
  justify-content: space-between;
  /* Options: flex-start | flex-end | center | space-around | space-evenly */

  /* Vertical alignment */
  align-items: center;
  /* Options: flex-start | flex-end | center | stretch */

  /* Wrap onto next line if needed */
  flex-wrap: wrap;

  /* Gap between items */
  gap: 1rem;
}

CSS Grid

CSS Grid is perfect for two-dimensional layouts â€” think page templates, image galleries, and card grids. While Flexbox works in one direction (row or column), Grid works in both.

style.css
.grid-container {
  display: grid;

  /* 3 equal columns */
  grid-template-columns: 1fr 1fr 1fr;

  /* Shorthand: repeat(3, 1fr) = same as above */
  grid-template-columns: repeat(3, 1fr);

  /* Auto-fill: as many columns as fit */
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

  gap: 1.5rem;
}

/* Make an item span multiple columns */
.featured-card {
  grid-column: span 2;
}
ðŸ’¡ Flexbox vs Grid â€” When to Use Which?

Use Flexbox for one-dimensional layouts (a navigation bar, a row of buttons). Use Grid for two-dimensional layouts (page structure, card galleries, complex templates).

Responsive Design

Responsive design means your website looks great on all screen sizes â€” from phones to tablets to desktops. The key tool is media queries.

style.css
/* Default styles (mobile first) */
.card-grid {
  grid-template-columns: 1fr;  /* 1 column on mobile */
}

/* Tablet: screens 768px and wider */
@media (min-width: 768px) {
  .card-grid {
    grid-template-columns: 1fr 1fr;   /* 2 columns */
  }
}

/* Desktop: screens 1024px and wider */
@media (min-width: 1024px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr);  /* 3 columns */
  }
}

/* Responsive font sizes with clamp() */
h1 {
  /* min size, ideal, max size */
  font-size: clamp(1.8rem, 4vw, 3.5rem);
}

Transitions & Animations

CSS can animate elements without any JavaScript! Use transitions for simple hover effects, and keyframes for complex animations.

style.css
/* TRANSITIONS â€” smooth change on hover */
.btn {
  background: purple;
  transition: all 0.3s ease;  /* what, duration, easing */
}
.btn:hover {
  background: darkviolet;
  transform: translateY(-4px);  /* move up on hover */
  box-shadow: 0 8px 30px rgba(128,0,128,0.4);
}

/* KEYFRAME ANIMATIONS */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

.animated-box {
  animation: fadeIn 0.6s ease both;
  /* name  duration  easing  fill-mode */
}

.spinner {
  animation: spin 1s linear infinite;
}
