Why Java?

Java was created in 1995 and is still one of the most widely used languages on Earth. Its motto ‚Äî "Write Once, Run Anywhere" ‚Äî means Java code runs on any device with a Java Virtual Machine (JVM).

Java PowersExamples
Android DevelopmentAlmost all Android apps were built with Java or Kotlin
Enterprise BackendsSpring Boot ‚Äî used by Netflix, LinkedIn, Uber
Banking & FinanceMost banking software runs on Java
Big DataHadoop, Apache Spark are Java-based
Web ServersApache Tomcat, Spring MVC
Desktop AppsIntelliJ IDEA, Minecraft are built in Java
‚òï Java vs JavaScript

Despite similar names, Java and JavaScript are completely different languages. Java is strongly typed and compiled; JavaScript runs in browsers. The name similarity is purely historical marketing.

Program Structure

Every Java program lives inside a class, and execution begins at the main method. This might look verbose at first ‚Äî but it will make sense once you learn OOP.

HelloWorld.java
// Every Java file has one public class matching the filename
public class HelloWorld {

    // main() is the entry point ‚Äî Java starts here
    public static void main(String[] args) {

        // Print to the console
        System.out.println("Hello, World!");
        System.out.println("Welcome to NkTechVision!");

        // print() ‚Äî no newline at end
        System.out.print("Java ");
        System.out.print("is powerful!");

        // printf() ‚Äî formatted output
        System.out.printf("Name: %s, Age: %d%n", "Kishor", 25);
    }
}
Output
Hello, World! Welcome to NkTechVision! Java is powerful! Name: Kishor, Age: 25
üí° Java is Case-Sensitive

String and string are different. Class names start with a capital letter (PascalCase). Method and variable names use camelCase. Constants use ALL_CAPS.

Variables & Data Types

Java is statically typed ‚Äî you must declare the type of every variable. The compiler catches type errors before your program even runs.

DataTypes.java
public class DataTypes {
    public static void main(String[] args) {

        // Primitive types
        int age = 25;               // whole number  (-2B to 2B)
        long population = 8000000000L; // large integer
        double pi = 3.14159;        // decimal (64-bit)
        float price = 9.99f;         // decimal (32-bit)
        char grade = 'A';            // single character
        boolean isFree = true;      // true or false
        byte level = 127;            // -128 to 127

        // Reference type
        String name = "NkTechVision"; // text (object)

        // var ‚Äî type inference (Java 10+)
        var city = "Mumbai";          // compiler infers String

        // Constants ‚Äî cannot be changed
        final int MAX_SCORE = 100;

        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Free: " + isFree);
    }
}

Input & Output

Java uses the Scanner class to read user input from the console. You need to import it first.

UserInput.java
import java.util.Scanner;  // Must import Scanner

public class UserInput {
    public static void main(String[] args) {

        // Create a Scanner object
        Scanner sc = new Scanner(System.in);

        System.out.print("Enter your name: ");
        String name = sc.nextLine();  // read a full line

        System.out.print("Enter your age: ");
        int age = sc.nextInt();       // read an integer

        System.out.println("Hello, " + name + "!");
        System.out.println("In 5 years you'll be " + (age + 5));

        sc.close();  // Always close the scanner
    }
}
üí° Scanner Methods

nextLine() reads a whole line of text. nextInt() reads an integer. nextDouble() reads a decimal. next() reads one word. Always close the scanner with sc.close() when done.

Operators.java
int a = 15, b = 4;

System.out.println(a + b);   // 19
System.out.println(a - b);   // 11
System.out.println(a * b);   // 60
System.out.println(a / b);   // 3  (integer division!)
System.out.println(a % b);   // 3  (remainder)

// Integer division vs double division
System.out.println(15 / 4);         // 3
System.out.println(15.0 / 4);       // 3.75
System.out.println((double) 15 / 4); // 3.75

// Ternary operator
int age = 20;
String status = (age >= 18) ? "Adult" : "Minor";
System.out.println(status);  // Adult
‚ö†Ô∏è Integer Division Trap

In Java, 15 / 4 gives 3, not 3.75! When both numbers are integers, Java performs integer division and truncates the decimal. Cast to double: (double) 15 / 4 to get 3.75.

If / Else / Switch

Conditionals.java
int score = 78;

// if / else if / else
if (score >= 90) {
    System.out.println("Grade: A");
} else if (score >= 70) {
    System.out.println("Grade: B");  // ‚Üê this runs
} else if (score >= 50) {
    System.out.println("Grade: C");

    System.out.println("Grade: F");

// switch statement ‚Äî great for exact values

    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break; // runs

}

// Modern switch expression (Java 14+)
String dayName = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3 -> "Wednesday";
    default -> "Unknown";
};

Loops

Loops.java
// FOR loop
for (int i = 1; i <= 5; i++) {
    System.out.println("Count: " + i);
}

// WHILE loop
int n = 1;
while (n <= 5) {
    System.out.println("n = " + n);
    n++;
}

// DO-WHILE ‚Äî runs at least once
int x = 10;
do {
    System.out.println("x = " + x);
    x++;
} while (x < 10);  // prints once even though false

// Enhanced for loop (for-each) ‚Äî loop through arrays
String[] courses = {"HTML", "CSS", "Java"};
for (String course : courses) {
    System.out.println("Course: " + course);
}

// break and continue
for (int i = 0; i < 10; i++) {
    if (i == 3) continue;  // skip 3
    if (i == 7) break;    // stop at 7
    System.out.print(i + " ");
}

Methods

In Java, functions are called methods and must be defined inside a class. Every method has a return type, a name, and optional parameters.

Methods.java
public class Methods {

    // void = returns nothing
    public static void greet(String name) {
        System.out.println("Hello, " + name + "!");
    }

    // int = returns an integer
    public static int add(int a, int b) {
        return a + b;
    }

    // Method overloading ‚Äî same name, different params
    public static double add(double a, double b) {
        return a + b;
    }

    // Recursive method
    public static int factorial(int n) {
        if (n <= 1) return 1;
        return n * factorial(n - 1);
    }

    public static void main(String[] args) {
        greet("Kishor");                  // Hello, Kishor!
        System.out.println(add(5, 3));    // 8
        System.out.println(add(2.5, 3.5)); // 6.0
        System.out.println(factorial(5)); // 120
    }
}
üîë Method Overloading

Java allows multiple methods with the same name as long as their parameter types or count differ. The compiler picks the right one based on what arguments you pass. This is called method overloading.

Arrays

An array in Java stores a fixed-size sequence of elements of the same type. The size cannot change after creation.

Arrays.java
// Declare and initialise
int[] scores = {95, 87, 76, 91, 83};
String[] subjects = new String[3]; // empty array of size 3
subjects[0] = "HTML";
subjects[1] = "CSS";
subjects[2] = "Java";

// Access and length
System.out.println(scores[0]);     // 95
System.out.println(scores.length); // 5

// Loop through array
for (int i = 0; i < scores.length; i++) {
    System.out.println("Score " + i + ": " + scores[i]);
}

// For-each loop (cleaner)
for (String sub : subjects) {
    System.out.println(sub);
}

// Sort an array
java.util.Arrays.sort(scores);
System.out.println(java.util.Arrays.toString(scores));
// [76, 83, 87, 91, 95]

// 2D array (like a table)
int[][] matrix = {{1,2,3}, {4,5,6}, {7,8,9}};
System.out.println(matrix[1][2]);  // 6

Strings

In Java, String is an object (not a primitive). It comes with many useful built-in methods.

Strings.java
String s = "NkTechVision";

// Length
System.out.println(s.length());       // 12

// Upper / Lower case
System.out.println(s.toUpperCase()); // NKTECHVISION
System.out.println(s.toLowerCase()); // nktechvision

// Contains, starts, ends
System.out.println(s.contains("Tech"));    // true
System.out.println(s.startsWith("Nk"));  // true
System.out.println(s.endsWith("Vision")); // true

// Replace
System.out.println(s.replace("Nk", "NkTech"));

// Substring
System.out.println(s.substring(0, 2));  // "Nk"
System.out.println(s.substring(2));     // "TechVision"

// Trim whitespace
String padded = "  hello  ";
System.out.println(padded.trim());  // "hello"

// Compare strings (use .equals(), NOT ==)
String a = "Java", b = "Java";
System.out.println(a.equals(b));           // true ‚úì
System.out.println(a.equalsIgnoreCase(b)); // true

// String.format() ‚Äî like printf but returns a String
String msg = String.format("Score: %d / %d", 89, 100);
System.out.println(msg);  // Score: 89 / 100
‚ö†Ô∏è Always use .equals() to compare Strings

Never use == to compare String values in Java. == checks if they're the same object in memory. Use .equals() to compare the actual text content.

Classes & Objects

Java is built around Object-Oriented Programming. A class is a blueprint; an object is a specific instance created from that blueprint.

Student.java
public class Student {

    // Fields (instance variables)
    private String name;
    private int age;
    private double gpa;

    // Constructor ‚Äî runs when object is created
    public Student(String name, int age, double gpa) {
        this.name = name;
        this.age = age;
        this.gpa = gpa;
    }

    // Getters (read private fields)
    public String getName() { return name; }
    public int    getAge()  { return age; }

    // Setter (update a field with validation)
    public void setGpa(double gpa) {
        if (gpa >= 0 && gpa <= 10) this.gpa = gpa;
    }

    // toString ‚Äî how object prints
    @Override
    public String toString() {
        return String.format("Student{name='%s', age=%d, gpa=%.1f}",
                             name, age, gpa);
    }
}

// In main() / another class:
Student s1 = new Student("Kishor", 25, 8.5);
Student s2 = new Student("Priya", 22, 9.2);

System.out.println(s1);            // Student{name='Kishor', ...}
System.out.println(s2.getName()); // Priya

OOP Concepts

Java is built on four pillars of OOP. These concepts are what make Java code scalable, maintainable, and reusable.

Encapsulation
Hide internal data with private. Expose it safely with getters/setters. Protects your data from misuse.
Inheritance
A child class extends a parent, inheriting its methods and fields. Avoids code duplication.
Polymorphism
Same method name, different behaviour depending on the object. Achieved via method overriding.
Inheritance.java
// Parent class
public class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }

    public void speak() {
        System.out.println(name + " makes a sound");
    }
}

// Child class ‚Äî extends Animal
public class Dog extends Animal {

    public Dog(String name) {
        super(name);  // call parent constructor
    }

    @Override          // Polymorphism!
    public void speak() {
        System.out.println(name + " says: Woof!");
    }
}

// Usage
Animal a = new Animal("Animal");
Dog    d = new Dog("Bruno");

a.speak();  // Animal makes a sound
d.speak();  // Bruno says: Woof!
