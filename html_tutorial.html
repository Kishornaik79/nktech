Introduction to HTML

HTML (HyperText Markup Language) is the backbone of every webpage on the internet. In this lesson you'll learn what HTML is, how it works, and write your very first webpage ‚Äî no experience needed.

What is HTML?

HTML is a markup language ‚Äî it describes the structure of a webpage using special codes called tags. Think of it like the skeleton of a building: HTML defines what's there (headings, paragraphs, images, links), while CSS (coming later) handles how it looks.

‚ÑπÔ∏è Did You Know?

HTML was invented by Tim Berners-Lee in 1991. The latest version is HTML5, which is what all modern browsers use today.

Every webpage you visit is built on HTML. When you "View Page Source" in your browser, you're reading HTML!

Basic Page Structure

Every HTML document follows the same basic structure. Here's a complete template:

HTML 
<!-- This tells the browser it's an HTML5 document -->
<!DOCTYPE html>

<html lang="en">

  <head>
    <!-- Invisible info about your page -->
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>

  <body>
    <!-- Everything visible goes here -->
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>

</html>
üí° Pro Tip

Notice how every opening tag like <body> has a matching closing tag </body>. The closing tag always has a forward slash /.

Attributes

Attributes give extra information to HTML tags. They're written inside the opening tag as name="value" pairs.

HTML
<!-- href tells the link WHERE to go -->
<a href="https://example.com" target="_blank">Open in new tab</a>

<!-- src is the image source, alt is for accessibility -->
<img src="cat.jpg" alt="A cute cat" width="300">

<!-- id and class are used to style elements with CSS -->
<p id="intro" class="highlight">Styled paragraph</p>

Links & Images

Links and images are the two most-used elements in any webpage. Here's how to use them properly:

Links

HTML
<!-- External link (opens in new tab) -->
<a href="https://google.com" target="_blank">Go to Google</a>

<!-- Internal link (goes to another page on your site) -->
<a href="about.html">About Me</a>

<!-- Link that scrolls to a section on the same page -->
<a href="#contact">Jump to Contact</a>

Lists

HTML has two types of lists: unordered (bullet points) and ordered (numbered).

HTML
<!-- Unordered list (bullet points) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered list (numbered) -->
<ol>
  <li>Open VS Code</li>
  <li>Write your HTML</li>
  <li>Open in browser</li>
</ol>

Semantic HTML

Semantic tags describe the meaning of content, not just its appearance. This helps search engines (like Google) and screen readers understand your page better.

HTML
<header>  <!-- Top section: logo, navigation -->
  <nav>...</nav>
</header>

<main>   <!-- Main content of the page -->
  <article>
    <h2>Blog Post Title</h2>
    <p>Content goes here...</p>
  </article>

  <aside>Sidebar info</aside>
</main>

<footer> <!-- Bottom of page: copyright, links -->
  <p>¬© 2025 NkTechVision</p>
</footer>
